# Weekly Report System

A full-stack web application that allows team members to submit weekly reports and allows managers to analyze team activity through a dashboard.

## Features

### Authentication
- User registration
- Login/logout
- JWT authentication
- Role-based access

Roles:
- Member
- Manager

### Team Member Features
- Create weekly reports
- Edit reports
- Delete reports
- View report history

Report fields:
- Week/date range
- Project/category
- Tasks completed
- Tasks planned
- Blockers
- Hours worked
- Notes

### Manager Features
- View all team reports
- Filter reports
- Dashboard analytics
- Work hour charts
- Submission tracking

### Project Management
- Create projects
- Update projects
- Delete projects
